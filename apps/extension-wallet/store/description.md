## Full Description

Ancore Wallet is a secure browser wallet for Stellar that brings account abstraction to everyday Web3 usage.

With Ancore, users can move from seed-phrase-heavy workflows to a safer, smoother wallet experience:

- Session keys for limited, time-bound signing permissions
- Clear onboarding and wallet setup flow
- Fast lock/unlock access controls
- Simple send and receive UX for XLM transfers
- Dedicated session key management for safer automation

Built for both new and advanced users, Ancore Wallet focuses on practical security:

- Wallet state is stored locally in browser extension storage
- Access can be locked and unlocked directly from the wallet UI
- Session key permissions are explicit and user-controlled
- The extension requests only minimal required permissions

Whether you are exploring Stellar for the first time or managing advanced account workflows, Ancore Wallet helps you transact with confidence and clarity.
